def patient_specific(patient_queryset=None, nicotine_abuse=None, hpv_status=None, neck_dissection=None, institution__in=None, **rest):

Filter QuerySet of patients based on patient-specific properties.

This function is designed in such a way that one can simply add another argument to its definition without actually changing the logic inside it and it will the be able to filter for that added argument (given that it is also a field in the model patients.models.Patient).

def tf2arr(value):

Map True, False & None to one-hot-arrays of length 3. This particular mapping comes from the fact that in the form True, None, False are represented by integers 1, 0, -1. So, the one-hot encoding uses an array of length 3 that is one only at these respective indices, where -1 is the last item.
