Here we define custom model fields that add some functionality to the existing Django model fields.

Class FieldFileWithHash FieldFile that adds an MD5 hash value as the property md5_hash to its defintion which in turn allows me to detect and block changes to the file and/or the dataset using that hash value.
Class FileFieldWithHash A normal FileField that uses my custom FieldFileWithHash instead of Django's built-in FieldFile.
Class RobustDateField DateField that doesn't raise a ValidationError when the date string isn't formated according to ISO (YYYY-MM-DD)
Exception CorruptedFileError Raised when FieldFileWithHash notices a file has been changed after its hash has been computed and fixed.
Exception DuplicateFileError Exception raised when e.g. during uploading or saving the Dataset model notices a new file is already stored in one of the other datasets.
