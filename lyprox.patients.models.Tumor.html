lyprox.patients.models.Tumor

class documentation

class Tumor(mixins.LockedDatasetMixin, loggers.ModelLoggerMixin, models.Model):

View In Hierarchy

Model to describe a patient's tumor in detail. It is connected to a patient via a django.db.models.ForeignKey relation.
